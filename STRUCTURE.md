# Project Structure

This file describes the structure and purpose of each major module and package in ProtoMind-v3.

- core/: Cognitive architecture modules
- environment/: Environment interface and simulation
- task/: Task abstraction, planning, and tool evolution
- main.py: Entry point for running ProtoMind
- requirements.txt: Python dependencies
- test_tool_evolution.py: Test suite for tool evolution

See CHANGELOG.md for recent changes and additions.
